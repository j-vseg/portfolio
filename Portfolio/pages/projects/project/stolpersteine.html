Stolpersteine

Android Application

Go on a walk with Stolpersteine!

Stolpersteine is a virtual map that show you the location of all the Stolpersteine in the Benelux. The app is a remodeled and renewed version of the original Stolpersteine.app

Check out code

Client: Frank van Dijk

Duration project: 3 weeks

Features

  • Show Stolpersteine on map
  • Display a list of Stolpersteine
  • Be able to search Stolpersteine and group them by People, Street and City
  • Clicking on a marker navigates to a detailed information page
  • Multi language (NL/EN/FR)
  • Your able to switch between languages
